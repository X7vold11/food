# 📱 Cara Membuat Icon PWA

Aplikasi PWA membutuhkan icon dalam berbagai ukuran. Berikut cara membuatnya:

## Opsi 1: Menggunakan Tool Online (TERMUDAH) ⭐

1. **Buka**: https://www.pwabuilder.com/imageGenerator
2. **Upload** gambar icon Anda (minimal 512x512px)
3. **Download** semua icon yang di-generate
4. **Upload** semua file PNG ke repository GitHub

## Opsi 2: Menggunakan File HTML Generator

1. **Buka file** `create-icons.html` di browser
2. **Klik** tombol "Generate Icons"
3. **Download** setiap icon dengan klik tombol download
4. **Upload** semua file PNG ke repository GitHub

## Opsi 3: Menggunakan Canva atau Photoshop

1. **Buat desain** icon 512x512px dengan:
   - Background: Gradient ungu (#667eea ke #764ba2)
   - Icon: Garpu dengan makanan
   - Text: "Cal" di bawah
2. **Export** dalam ukuran: 72, 96, 128, 144, 152, 192, 384, 512
3. **Simpan** dengan nama: `icon-72x72.png`, `icon-96x96.png`, dll
4. **Upload** ke repository GitHub

## Opsi 4: Menggunakan Icon Sederhana (CEPAT)

Gunakan emoji atau icon sederhana:
1. Buka https://favicon.io/emoji-favicons/
2. Pilih emoji 🍽️ atau 🥗
3. Generate dan download
4. Rename sesuai ukuran yang dibutuhkan
5. Upload ke repository

## Setelah Icon Siap:

1. Upload semua icon ke root folder repository
2. Commit dan push ke GitHub
3. PWA siap digunakan!

## Tips:

- Gunakan desain yang simple dan recognizable
- Pastikan icon terlihat jelas di ukuran kecil
- Gunakan warna yang kontras
- Test di berbagai device
